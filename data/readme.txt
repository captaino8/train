
你好欢迎使用鸿基梦余票提醒系统

环境安装与部署说明：

本系统使用者需要了解node最简单的操作(前端必备)
1. 下载安装 git 与 node
2. 下载代码压缩包或者使用 git clone https://gitee.com/hjm100/node_12306
3. cnpm 或者npm安装node依赖【详细可查看README.md】 npm install 或cnpm install 
    |——注：如果npm安装缓慢或者安装出错请使用 cnpm
    |——如何使用cnpm： 
    |——全局安装淘宝镜像
    |——$ npm install -g cnpm --registry=https://registry.npm.taobao.org
    |——使用cnpm 安装模块即可
4. node app开启服务【早上7:00:30后每隔一个小时自动帮你查票】
5. 你可以根据自己的需求去改变定时器

邮件备注：
本系统可以设置理想车次【邮件列表粉红色显示为理想车次】
